# CN Inventive Step Quick Guide

Use this guide for China inventive-step conclusions.

## Three-Step Method

1. Determine the closest prior art.
2. Identify distinguishing features and the actual technical problem.
3. Decide whether the claimed solution would have been obvious to the skilled person.

## Practical Rules

- Keep D1 selection stable before discussing motivation.
- Derive the objective technical problem from the differences and actual effect, not from hindsight.
- Check whether D2/D3 or common knowledge provides a real technical teaching, not just thematic similarity.
- Unexpected technical effect can materially strengthen inventiveness if it is evidenced.

## Common Traps

- treating the invention summary as the objective problem
- using “common general knowledge” with no proof
- ignoring teaching away or technical prejudice
- collapsing all differences into one vague conclusion

## Output Notes

State:

- D1 choice and why it is closest
- distinguishing features
- objective technical problem
- motivation strength and counterpoints
