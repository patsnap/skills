# 功能导向检索skill：利用此技能帮助研发人员寻找跨领域技术方案

## 概述

用户：技术研发人员 场景：有一个具体的技术问题，想要借鉴跨领域技术方案，不知道怎么去找，如何去找，去哪里找 【输入】一个特定的技术问题：例如如何提高手机散热能力 【过程】问题功能化、功能一般化、一般化专利检索、方案筛选、方案推荐 【输出】html报告。

## 触发场景

- 请帮我构建功能导向检索skill：利用此技能帮助研发人员寻找跨领域技术方案的检索策略。
- 请输出检索式、主体归并规则和结果清单。

## 核心能力

- **功能化技术问题**：把具体研发问题抽象成可跨领域搜索的一般功能表达。
- **寻找跨领域方案**：基于一般化功能检索其他行业可借鉴的技术方案。
- **筛选可迁移路径**：判断外部方案与当前场景的适配性、约束和改造空间。

## 输出内容

- **功能导向检索报告**：输出问题功能化过程、检索策略和推荐方案。
- **跨领域方案清单**：整理可借鉴专利或技术方案及其对应功能。
- **迁移应用建议**：说明方案迁移条件、潜在风险和下一步验证动作。

## 使用注意事项

- 不要编造不存在的专利、市场或技术证据。
- 优先使用用户提供材料和 PatSnap/Zhihuiya MCP 数据。
- 明确标注数据来源、检索范围、时间范围和不确定性。
- 涉及 FTO、侵权、有效性等法律判断时，不替代律师意见。
