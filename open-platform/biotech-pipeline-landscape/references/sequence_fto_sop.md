# 序列查询与FTO分析标准操作规程（SOP）

## 适用范围
CAR-T / 抗体类生物治疗产品的 scFv 序列专利侵权扫描（FTO）、竞品序列比对、新颖性核查、制造质控和患者回输前序列溯源。

---

## SOP 8步流程

### Step 1：输入序列准备

格式：FASTA格式，逐片段提交

片段提交优先级：
1. CD19 scFv VH（重链可变区）
2. CD19 scFv VL（轻链可变区）
3. CD22 scFv VH
4. CD22 scFv VL
5. CDR1 / CDR2 / CDR3（单独精准比对）
6. 全CAR构建体（氨基酸全序列）

### Step 2：数据库选择

| 数据库 | 适用场景 |
|--------|---------|
| ALLPATENT | 综合FTO扫描，全面覆盖 |
| CLAIMS | 高精度FTO核查，仅权利要求序列 |

推荐：先用 ALLPATENT 宽泛扫描，再用 CLAIMS 精准确认。

### Step 3：工具调用（提交序列搜索任务）

工具：ls_sequence_search_submit
参数：
- sequence_type: PROTEIN
- query_type: PROTEIN
- database: ["ALLPATENT"]
- sequence: FASTA格式序列
- evalue: 0.001
返回：job_id

### Step 4：状态轮询

工具：ls_sequence_search_check_status
参数：job_id
状态：TODO → RUNNING → SUCCESS
轮询间隔：30秒，最大等待：10分钟

### Step 5：阈值设置与结果获取

工具：ls_sequence_search_get_results
参数：
- perc_identity: {"start": 80}（≥80%）
- qcov_hsp_perc: {"start": 90}（≥90%）
- sort_field: "score", order: "DESC"

阈值调整：
| 场景 | identity | qcov |
|------|----------|------|
| 全面风险扫描（宽）| ≥70% | ≥80% |
| 标准FTO（推荐）| ≥80% | ≥90% |
| 高精度确认（严）| ≥90% | ≥95% |

### Step 6：结果分层判读

| 风险等级 | identity | CDR3 | 权利要求 |
|---------|----------|------|---------|
| 高风险 | ≥95% | 高度一致 | 命中有效claim |
| 中风险 | 80–95% | 部分差异 | 可能命中 |
| 低风险 | <80% | 显著不同 | 未命中 |

**免责提示**：
- 单个CDR残基差异不足以单独支撑独立IP空间判断
- FTO判断须同时考量：序列差异 + 构型差异 + 亲和力 + 功能数据
- 最终FTO结论须由专利律师确认

### Step 7：专利映射（FTO映射表必含字段）

| 字段 | 示例 |
|------|------|
| 查询片段 | CD19 VH / CD22 VL 等 |
| 命中专利号 | US20230241212A1（带链接）|
| 命中权利要求 | Claim 1, Claim 5 |
| 序列同一性 | 96.3% |
| 查询覆盖度 | 98.5% |
| CDR3差异 | 第99位 Y→F |
| 法律状态 | active |
| 有效地域 | US / EP / CA |
| 风险等级 | 高风险 |
| 绕开建议 | CDR3第99位保留Y，调整FR2两处残基 |

### Step 8：输出交付物清单

1. 命中清单（专利号 + 申请人 + 法律状态 + 风险等级）
2. CDR差异表（CDR1/2/3逐位比对，IMGT编号）
3. FTO风险矩阵（行：查询片段，列：专利号，颜色分级）
4. 设计绕开建议

设计绕开策略：
| 策略 | 操作 | 适用场景 |
|------|------|---------|
| 人源化框架区优化 | FR1-FR4人源化改造 | 绕开鼠源专利 |
| CDR移植变体 | CDR3≥2关键残基替换 | 新颖性区分 |
| 全人源单域抗体（VHH）| 纳米抗体框架 | 完全独立IP空间 |
| 膜近端表位定向 | CD22膜近端域 | 特异位点布局 |

## 制造QC序列检测方法

| 方法 | 检测节点 | 灵敏度 | 用途 |
|------|---------|--------|------|
| Sanger | 载体构建 | 中（>20%）| 单片段快速确认 |
| NGS | 病毒载体生产 | 高（<1%）| 低频突变、重排 |
| ddPCR | 产品放行 + 体内 | 极高（<0.01%）| VCN精确定量 |
| qPCR | 大批量筛查 | 高（<0.1%）| CAR-T扩增动力学 |
| 流式细胞仪 | 放行 + 体内 | 中（≥0.1%）| CAR表达率 |
| ISA | 安全性随访 | 高（单克隆）| 整合位点评估 |
