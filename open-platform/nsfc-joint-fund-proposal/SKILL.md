# 国家联合基金申报书撰写助手（NSFC-Joint-Fund-Proposal）

## 技能概述

本技能依照《2027年度国家联合基金指南编制模板》官方格式，辅助科研人员逐模块、有据可查地撰写 NSFC-湖北联合基金指南建议书（及类似省级联合基金申报书）。

支持以下核心场景：
- 给定研究方向/技术方案，自动生成完整申报草稿
- 针对某一板块（如创新点、关键科学问题）单独润色或扩写
- 检查现有文本是否符合官方模板要求
- 提供字数控制、术语规范、科学表达等专业建议

---

## 官方模板结构（必须严格遵循）

### 封面信息（元数据区）

| 字段 | 说明 | 约束 |
|------|------|------|
| 基金学部 | 单选：A数理、B化学、C生命、D地球、E工程与材料、F信息、G管理、H医学、T交叉 | 只能涉及一个科学部 |
| 学科代码 | 须填写到二级代码，如 F0402 | 必填 |
| 研究领域关键词 | 3-5个 | 必填 |
| 科学问题属性 | 单选：A鼓励探索突出原创 / B聚焦前沿独辟蹊径 / C需求牵引突破瓶颈 / D共性导向交叉融通 | 单选 |
| 指南需求 | 单选或多选：A国家科技创新战略部署 / B湖北重点产业前沿技术探索 / C湖北高端人才前期培育 / D双一流学科建设引领升级 | 可多选 |
| 指南研发需求是否来自在鄂企业 | 是 / 否 | 必填 |

### 建议书正文结构

#### 第一部分：基本信息
- 建议指南名称（不超过50字）
- 所属领域（单选：电子信息 / 生物与农业 / 环境与生态 / 新材料与先进制造 / 人口与健康 / 能源与化工 / 现代交通与航空航天 / 海洋科学）
- 建议单位、依托平台名称与类型（选填）
- 建议人（姓名、职务/职称）
- 意向申报人（姓名、职务/职称）
- 单位及通讯地址、邮编、E-mail、手机号码

#### 第二部分：指南说明（不超过150字）
标准句式：以XX为研究对象，开展XX应用基础研究，解决XX的关键科学问题，为XX提供依据。

写作要点：
- 精炼表达研究对象、研究类型、核心科学问题、应用价值
- 严格控制在150字以内
- 语言简洁、逻辑清晰

#### 第三部分：主要研究内容（正文不超过1500字）

必须覆盖的子板块：
1. 核心研究内容（分条列出，每条清晰标注序号）
2. 研究背景与领域问题（国内外卡脖子现状）
3. 科学意义（理论价值与实践价值）
4. 国内外研究概况和发展趋势

写作规范：
- 分条编号，每条独立成意
- 研究内容应直接对应研究目标与关键科学问题
- 强调突破点与技术路线的新颖性

#### 第四部分：建议作为联合基金项目指南的理由

必须包含以下四个子板块：
1. 领域背景：阐述研究对象的重要性、行业痛点、卡脖子问题
2. 科学意义：理论与应用双重价值
3. 国内外研究概况：客观描述国内外进展、差距与空白
4. 指南需求类型说明：说明所选需求类型的匹配逻辑

#### 第五部分：建议研究内容和目标 / 关键科学问题 / 创新点 / 预期成果

建议研究内容：
- 以体系构建为主线，分阶段、分层次描述
- 融合基础研究与应用验证

建议研究目标：
- 目标须可量化、可验证
- 与研究内容一一对应

拟解决关键科学问题（通常3-5个）：
- 每个问题独立表述
- 语言须体现科学挑战而非技术任务

创新点（通常4-6条）：
- 每条须包含：创新维度（体系/技术/理论/方法）+ 突破内容 + 首创性声明
- 标准句式：首次提出/建立/实现……，突破……局限，为……提供……

预期研究成果：
- 学术成果（SCI论文数量及档次）
- 知识产权成果（专利数量）
- 平台/规程/标准成果（SOP、数据库等）

湖北应用前景（按指南需求分类描述）：
- 产业应用（生物医药、大健康等）
- 临床应用（医院、诊疗中心）
- 科研与平台建设
- 产业化落地前景

#### 第六部分：国内已具备条件和基础

必须覆盖：
- 国内外研究团队进展
- 现有实验条件
- 可牵头申报的研究团队
- 可依托的国家级平台
- 联合申报力量
- 本单位优势与劣势客观分析

#### 第七部分（选填）：指南研发需求来源企业基本情况

如需求来自企业，填写企业基本情况；否则可略过。

#### 第八部分：国家自然科学基金委及其他部委资助情况

- 列举相关领域已获资助项目
- 说明本研究与现有资助的互补性

## 工作流程

### Step 1：信息收集
向用户确认以下必要信息：
1. 研究方向/技术方案（核心内容）
2. 所属科学部（A-T 单选）
3. 学科代码（二级代码）
4. 科学问题属性（A/B/C/D 单选）
5. 指南需求类型（A/B/C/D，可多选）
6. 是否来自在鄂企业
7. 所属领域（单选八选一）
8. 建议单位 / 建议人 / 意向申报人（如有）
9. 字数偏好（各板块默认按模板上限）

### Step 2：结构解析
基于用户提供的研究方向，自动提炼：
- 技术三要素（技术问题 / 技术方案 / 技术效果）
- 核心创新点（体系/技术/理论/方法维度）
- 关键科学问题（科学挑战，而非工程任务）
- 可量化研究目标

### Step 3：逐模块生成
按官方模板顺序逐模块输出，每模块输出后询问是否需要调整，再生成下一模块。

### Step 4：格式审查
- 字数检查（指南说明不超过150字，正文不超过1500字）
- 标准用语合规检查
- 逻辑连贯性检查（研究内容和目标和关键科学问题和创新点相互对应）

### Step 5：完整草稿输出
汇总所有板块，输出可直接粘贴进Word的完整申报草稿。

## 写作规范与质量标准

### 语言规范
- 使用学术专业语言，避免口语化表达
- 科学问题描述须体现"未知性"与"挑战性"
- 创新点须包含首创性声明（"国内外首次"/"首次提出"/"首次实现"等）
- 避免使用模糊用词（"相关""一定程度上"等）

### 字数控制
| 板块 | 字数限制 |
|------|---------|
| 指南名称 | 不超过50字 |
| 指南说明 | 不超过150字 |
| 主要研究内容正文 | 不超过1500字 |
| 各研究目标条目 | 建议每条20-50字 |
| 创新点每条 | 建议80-150字 |

### 科学问题质量标准
合格的关键科学问题须满足：
- 聚焦"未知机制"而非"技术任务"
- 可被实验/理论研究回答
- 与研究内容和创新点直接对应
- 具有学科前沿代表性

---

## 常见错误与规避

| 常见问题 | 规避方法 |
|---------|---------|
| 研究内容写成工作任务而非科学问题 | 将"完成XX系统搭建"改为"探究XX的调控机制" |
| 创新点缺乏首创性声明 | 每条以"首次/国内外首次"开头 |
| 指南说明超过150字 | 严格控制，去除修饰语 |
| 关键科学问题与创新点内容重叠 | 问题聚焦"未知"，创新点聚焦"突破" |
| 预期成果缺少知识产权和平台成果 | 补充专利数量和SOP/标准成果 |
| 湖北应用前景脱离地域特色 | 结合湖北具体产业链和机构具体描述 |

## 参考示例（基于官方模板中的体外人工结肠项目）

指南说明示例：
以体外人工结肠生产系统为研究对象，开展肠道功能菌群定向培育应用基础研究，解决"黑箱细菌"不可培养及FMT规模化生产的关键科学问题，为临床肠道菌群精准干预提供理论依据与技术产业支撑。（约95字）

创新点示例：
体系创新：国内外首次创建体外人工结肠生产系统连续发酵模型，实现肠道微生态动态模拟与长期稳态培养，突破静态培养无法复刻肠道蠕动、底物更替、菌群演替的局限，通过调整培养条件定向富集培养功能菌群。

关键科学问题示例：
肠道未培养微生物（黑箱细菌）不可培养瓶颈问题：传统纯培养无法实现肠道大量未培养菌活化增殖，如何依托结肠连续发酵的共培养体系，实现未培养黑箱细菌原位富集与定向活化，阐释混合菌群共生互作机制？
