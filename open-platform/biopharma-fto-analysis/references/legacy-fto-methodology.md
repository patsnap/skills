# FTO 战略分析 Skill

## 定位

本 Skill 将《抗体类药物FTO报告撰写操作手册》中的方法论固化为可复用的智能分析工作流。**产品/技术领域为可变参数**，支持单抗、多抗、ADC、小核酸、基因治疗、小分子等任意领域的 FTO 分析。

## 激活条件

当用户的查询涉及以下任意一项时，加载本 Skill：

- 专利自由实施（FTO / Freedom to Operate）分析
- 药物/产品上市前的专利风险评估
- 特定产品或技术的专利侵权风险排查
- FTO 报告撰写或 FTO 检索策略制定
- 仿制药/生物类似药的专利清障分析

**典型查询示例：**
- 帮我做替雷利珠单抗的 FTO 分析
- 我们的 ADC 药物准备进入美国市场，需要 FTO 报告
- 分析某小分子靶向药在中国的自由实施风险
- 帮我制定 PD-1 单抗的 FTO 检索策略

## 输入变量（必须向用户确认）

| 变量 | 说明 | 示例 |
|------|------|------|
| `PRODUCT_NAME` | 产品/药物名称或描述 | 替雷利珠单抗、DS-8201、自研ADC-001 |

## 工作流（7 阶段标准 FTO 流程）

### 阶段 0：技术解析与输入整理

**目标：** 理解产品技术特点，明确检索维度

**操作步骤：**
1. 调用 `ls_drug_fetch` 或 `ls_drug_search` 在新药情报库中检索产品基础信息
2. 获取：靶点信息、核心序列（如适用）、已知核心专利、合作伙伴、临床试验适应症
3. 根据 `TECH_DOMAIN` 确定技术解析维度：
   - **单抗：** CDR序列（HCDR1/2/3, LCDR1/2/3）、VH/VL序列、H链/L链全长、靶点、制剂、用途
   - **多抗/双抗：** 各结合臂分别解析（同单抗流程），抗体结构描述
   - **ADC：** 抗体部分（同单抗）+ 毒素化合物（SMILES结构）+ 连接子（SMILES结构）
   - **小分子：** SMILES结构、核心药效团、晶型
   - **小核酸：** 核苷酸序列、化学修饰位点、递送载体
4. 输出技术解析摘要，供用户确认

### 阶段 1：检索准备

**目标：** 确定检索范围和数据库策略

**操作步骤：**
1. **地域范围确定：**
   - 中国：CNIPA 授权专利 + PCT 进入国家阶段专利
   - 美国：USPTO 授权专利（含 PTA/PTE 延长）+ 橙皮书药品专利
   - 欧洲：EPO 欧洲专利（覆盖44个缔约/生效国）+ 各国国家专利
   - WIPO/PCT：注意"空窗期"问题（超30-32个月标记"失效"但仍在国家阶段形式审查中）
2. **时间范围：** 发明专利20年保护期 + 各国药品专利期限补偿（最长25年）
3. **有效性筛选：** 排除已确定驳回，保留复审中/行政诉讼中的专利
4. **检索数据库：** 智慧芽专利数据库（主）+ Bio序列数据库 + 新药情报库（辅）

### 阶段 2：核心专利检索（按技术领域执行）

**2A. 抗体类（单抗/多抗/ADC抗体部分）检索策略：**

**序列检索维度（优先级排序）：**

1. **CDR 序列检索（最高优先级）**
   - 调用 `ls_sequence_search_submit` 分别检索 HCDR1/2/3 和 LCDR1/2/3
   - 注意：分别筛选匹配1个、2个、3个CDR的专利（漏检风险点）
   - 不建议对不同CDR标注方式（IMGT/Chothia/Kabat/Martin）分别检索，以VH/VL互补

2. **VH/VL 可变区序列检索（补充）**
   - 相似度阈值：≥80%（识别"轻链可变区SEQ ID NO:X及80%以上同一性"类权利要求）
   - 调用 `ls_sequence_search_submit`

3. **H链/L链全长序列检索（精确）**
   - 采用精确检索，作为最小保护范围的底线核查
   - 调用 `ls_sequence_search_submit`

4. **公知公用序列识别：**
   - 建议使用 BLASTP/BLASTN 确认序列公知公用部分
   - 通过 NCBI 登录号判断公开时间（登录号靠后越新）

**结构/关键词检索维度：**

5. **靶点关键词检索**
   - 构建靶点 + 适应症 + 抗体关键词检索式
   - 调用 `mcp_patent-search__patsnap_search`，覆盖未限定序列的广谱权利要求

6. **制剂/配方检索**
   - 关键词：靶点 + 抗体 + 制剂/formulation/pharmaceutical composition

7. **联合用药/用途检索**
   - 关键词：靶点 + 联合用药搭档 + 适应症

8. **表达载体/生产平台检索**
   - 针对生产工艺，检索宿主细胞、表达系统、纯化方法相关专利

**ADC 附加检索维度：**

9. **毒素化合物结构检索**
   - 调用 `ls_structure_search`（EXT精确 + SIM相似性检索，阈值≥0.85）

10. **连接子结构检索**
    - 调用 `ls_structure_search`，覆盖各类可裂解/不可裂解连接子

**2B. 竞品企业全量专利检索：**
- 按 `COMPETITORS` 列表，调用 `ls_patent_search` 按申请人全面检索
- 扩展：母公司、子公司、技术受让方、许可方、核心发明人
- 调用 `ls_organization_pipeline_fetch` 获取竞品管线补充信息

**2C. CRISPR/基因编辑特别检索（如适用）：**
- 单独构建 CRISPR/Cas9 相关检索式，评估生产工艺侵权风险

### 阶段 3：专利筛选（两阶段法）

**初筛阶段（反向排除法）：**
- 排除明显不相关专利，相关性存疑全部保留
- 暂不分析保护范围和版本
- 产出：初筛专利列表

**精筛阶段：**
- 将权利要求技术方案与技术事实初步比对
- 同族拓展：在同一家族中确定保护范围最大专利
- 调用 `mcp_patent-search__patsnap_fetch` 获取全文权利要求详细分析
- 产出：高相关专利列表（含风险评级草案）

### 阶段 4：侵权判定分析

**适用原则：全面覆盖原则（逐一比对所有技术特征）**

对每一项高相关专利：

1. **技术特征逐一比对：**
   - 列出权利要求所有技术特征
   - 对应技术事实逐条比对
   - 判断：完全相同 / 等同 / 不落入

2. **特殊情形处理：**
   - 待分析方案包含专利技术特征 + 额外特征（情形②）→ 通常仍落入保护范围
   - 封闭式组合物权利要求 + 新增特征 D → 原则上不侵权，但需判断 D 是否实质改变性质
   - 等同特征认定：基本相同手段、基本相同功能、基本相同效果 + 无需创造性劳动

3. **风险等级评定：**
   - 🔴 **高风险**：极可能落入有效专利保护范围，且无明显规避路径
   - 🟡 **中风险**：有侵权可能，但存在规避方案或专利有效性存疑
   - 🟢 **低风险**：不落入保护范围，或专利已过期/驳回
   - ⚪ **观察**：在审专利，需持续跟踪

### 阶段 5：总结与建议

按以下结构输出：

**5.1 整体自由实施风险等级评定**
- 分市场（CN/US/EU/JP）给出风险评级
- 区分有效专利风险 vs 在审专利风险
- 给出整体评定结论

**5.2 应对策略建议**

针对有效专利：
- 专利无效请求（提出无效理由：新颖性/创造性/说明书支持）
- 合作/许可谈判
- 技术规避方案设计
- 等待专利过期

针对在审专利：
- 提交第三方意见（Third Party Observations）
- 阻止授权或缩小保护范围
- 持续监控审查动态

**5.3 跟踪建议**
- 关键专利到期时间表
- 在审专利授权动态监控节点
- 建议定期复查周期

**5.4 专利布局建议**
- 差异化专利申请方向（技术维度）
- 建议申请市场（区域维度）
- 申请时机建议（时间维度）

### 阶段 6：附录生成

输出标准附录模板：
1. **司法条款** — 《专利法》第五十九条、最高法司法解释、全面覆盖原则
2. **相关诉讼案例参考** — 同领域典型案例（如适用，调用 `ls_news_vector_search` 检索）
3. **免责声明** — 标准五条款模板

## 注意事项 & 质量控制

1. **PCT 空窗期**：超30-32个月标记"失效"不等于失效，需单独核查是否进入国家阶段
2. **CDR漏检**：权利要求单独保护1-2个CDR时，数据库高相关度可能偏低，需手动扩检
3. **同族拓展**：筛选时在同族内选保护范围最大专利进行分析
4. **国家专利期限计算**：美国 PTA+PTE、中国新药专利期限补偿，需个案核查
5. **欧洲专利**：EP专利在各成员国验证后生效，需区分验证国范围
6. **等同侵权**：在全面覆盖原则下还需考虑等同特征，不能仅做字面比对
7. **来源引用**：所有专利分析结论需附专利号和权利要求条款作为证据
