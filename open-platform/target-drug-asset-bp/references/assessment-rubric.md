# 成果转化成熟度评分标准 — university-tech-transfer-packager

满分 100 分。评分须基于实际工具检索到的证据，不得仅凭用户描述打分。

---

## 评分维度

### 1. Scientific Novelty 科学新颖性（15 分）

| 分数 | 标准 |
|---|---|
| 13–15 | 靶点/机制为首次报道或近 3 年新发现，在 ls_translational_medicine / ls_paper 中竞争数量极少（< 5 个同类研究），核心 claim 有同行评审论文支撑 |
| 9–12 | 靶点已知但亚型/通路/机制角度新颖，同类研究 5–20 个，有机制文献支撑 |
| 5–8 | 靶点和机制均已知，但技术路线（如给药方式/剂型/联合方案）有一定创新 |
| 1–4 | 靶点和技术路线均为成熟领域，创新点不明确 |
| 0 | 无法从现有材料判断新颖性 |

**评分依据**：ls_translational_medicine_search 返回数量 + ls_paper_search 核心论文质量 + 用户提供的核心 claim

---

### 2. Translational Evidence 转化证据强度（20 分）

| 分数 | 标准 |
|---|---|
| 17–20 | 有人类遗传学/基因组学证据（GWAS/孟德尔随机化），或有人体 PoC 数据（早期临床），并有多个独立来源验证 |
| 13–16 | 有体内动物模型（至少 2 个物种），有明确表型拯救/疾病相关功能证据，有 biomarker 假说 |
| 8–12 | 有体外细胞/类器官模型数据，1 个物种动物实验，转化链条基本清晰 |
| 3–7 | 仅有体外数据或计算预测，缺乏体内验证 |
| 0–2 | 仅有机制假说，无实验证据 |

**评分依据**：ls_translational_medicine_search + ls_clinical_trial_result_search 早期信号 + 用户提供实验描述

---

### 3. Clinical and Market Relevance 临床/市场相关性（15 分）

| 分数 | 标准 |
|---|---|
| 13–15 | 疾病负担高（大适应症或重大罕见病），当前治疗手段有明确局限，ls_clinical_guideline 确认未满足需求，国内有纳入医保/优先审评潜力 |
| 9–12 | 疾病负担中等，有一定未满足需求，市场空间可量化 |
| 5–8 | 疾病负担有限或市场竞争已饱和，但存在细分机会 |
| 1–4 | 罕见病且无孤儿药政策支持，或市场极小 |
| 0 | 无法判断临床或市场价值 |

**评分依据**：ls_clinical_trial_search 活跃度 + ls_clinical_guideline_vector_search + ls_drug_search 市场规模信号

---

### 4. Differentiation vs Competitors 差异化空间（15 分）

| 分数 | 标准 |
|---|---|
| 13–15 | 同赛道药物 < 5 个，无已获批竞品，或有明确 first-in-class / best-in-class 主张，且有机制/数据支撑 |
| 9–12 | 同赛道有竞品但本项目有清晰差异化角度（亚群/剂型/副作用/便利性） |
| 5–8 | 同赛道竞争较激烈，差异化点需进一步验证 |
| 1–4 | 同赛道已有多个获批竞品，差异化空间不明确 |
| 0 | 完全 me-too，无差异化点 |

**评分依据**：ls_drug_search 数量 + highest_phase 分布 + ls_clinical_trial_result_search 失败/成功信号

---

### 5. IP Readiness 专利与权属准备度（15 分）

| 分数 | 标准 |
|---|---|
| 13–15 | 已申请核心专利（中国 + PCT），权属清晰（机构持有），专利保护核心技术点，无明显 FTO 风险 |
| 9–12 | 已申请中国专利，权属基本清晰，PCT/海外布局待跟进，FTO 风险低–中 |
| 5–8 | 专利在准备中或仅有技术秘密保护，权属存在潜在争议（如校企合作项目），FTO 风险中 |
| 1–4 | 无已申请专利，核心技术已有他人专利覆盖，权属不清 |
| 0 | 关键信息缺失，无法判断 |

**评分依据**：ls_patent_search / ls_patent_fetch + 用户提供的专利信息 + 权属声明

---

### 6. Partnerability 企业合作适配度（10 分）

| 分数 | 标准 |
|---|---|
| 9–10 | 有 2 个以上明确匹配企业（ls_organization_pipeline_fetch 确认），合作模式清晰（授权/联合开发/技术入股），PI 有成果转化经验 |
| 6–8 | 有 1–2 个匹配方向，合作模式基本清晰，需要进一步接触 |
| 3–5 | 合作方向模糊，PI/机构无成果转化经验，需要中介平台 |
| 0–2 | 无明确合作候选方，技术路线与产业需求脱节 |

**评分依据**：ls_organization_pipeline_fetch + ls_drug_search 企业分布 + 用户提供的已有合作信息

---

### 7. Next-Step Feasibility 下一步验证可行性（10 分）

| 分数 | 标准 |
|---|---|
| 9–10 | 下一步实验明确（可在 12 个月内完成），成本可控，有可行的资金路径（基金/横向/转化基金），监管路径清晰 |
| 6–8 | 下一步方向清晰但时间线或资金有不确定性 |
| 3–5 | 下一步需要大量资源，路径不完全清晰 |
| 0–2 | 下一步高度不确定，或依赖突破性技术进展 |

**评分依据**：综合所有阶段信息 + 用户提供的研究计划

---

## 判断标签

| 标签 | 总分范围 | 含义 | 建议行动 |
|---|---|---|---|
| **Transfer Ready** | 80–100 | 材料相对完备，可进入企业 BD 或路演流程 | 准备路演材料，启动定向 BD 接触 |
| **Partnering Candidate** | 65–79 | 有较强转化潜力，但需补充若干关键证据 | 补充 1–2 项关键实验或专利，6 个月内可达 Transfer Ready |
| **Incubation Needed** | 45–64 | 基础扎实但转化成熟度不足，适合孵化阶段 | 进入孵化器，补实验、申专利、找种子基金 |
| **Research Only** | < 45 | 当前更适合发论文/申基金，不建议马上产业化 | 继续发表高质量论文，待转化证据积累后重新评估 |
| **Insufficient Materials** | — | 关键材料缺失（无法评分 ≥ 3 个维度） | 补充缺失信息后重新运行 |

## 评分原则

1. 有证据才打高分，不得仅凭用户描述给满分
2. 如某维度数据完全缺失，标注 `N/A（待补充）` 并在 Next-Step Package 中列为补充项
3. 各维度分项理由须简短但可验证（不超过 2 句话）
4. 总分四舍五入到整数
