# 园区产业精准招商报告

## 功能概述

为任意科技园区/产业园区生成一份《园区产业精准盘点、对标分析与智能招商策略报告》。

报告章节结构：
- 📋 执行摘要
- 🏭 第一章·产业全貌（自动化产业分类）
- 🌟 第二章·主导产业（综合优势指数评分）
- 🔍 第三章·对标分析（全国1-2个同类园区对标）
- 🔗 第四章·强链补链延链（策略建议）
- 🎯 第五章·精准招商清单（Top50企业 + 北京高校科技成果转化）

---

## 使用方式

直接告诉 Eureka：

> 帮我生成 **[园区名称]** 的产业招商报告

或使用更多自定义参数：

> 帮我生成 **[园区名称]** 的产业招商报告，主导产业聚焦 **[产业方向]**，对标园区选 **[对标园区]**，报告风格 **[紧凑/详细]**

---

## 输入参数

| 参数 | 是否必填 | 说明 | 示例 |
|------|---------|------|------|
| `park_name` | ✅ 必填 | 园区全称或简称 | 天津滨海中关村科技园 |
| `park_city` | 建议填写 | 园区所在城市/区域 | 天津市滨海新区 |
| `focus_industry` | 可选 | 指定重点关注产业方向 | 智能科技、生命大健康 |
| `benchmark_park` | 可选 | 指定对标园区 | 北京中关村科学城 |
| `report_style` | 可选 | 紧凑（领导版）/ 详细（研究版），默认紧凑 | 紧凑 |
| `data_cutoff` | 可选 | 数据截止日期，默认当前年月 | 2026年5月 |

## 执行流程

### 第一步：信息采集
1. 通过 `web.search` 联网搜索园区官网、管委会公告，获取：
   - 注册企业总数、注册资本规模
   - 园区主导产业定位（官方"N+X"产业方向）
   - 已入驻标杆企业名单
   - 所在城市产业政策与扶持政策
2. 调用智慧芽 PatSnap 企业数据工具（C011/C022/C007等）获取科创企业画像
3. 搜索对标园区公开数据（规模/结构/生态）

### 第二步：五章递进分析

**📋 执行摘要**
- 核心结论3-5条，KPI卡片展示关键数据
- 控制在1屏内可读完

**🏭 第一章·产业全貌**
- 按主标签（主产业）+ 副标签（细分方向）分类所有企业
- 输出：产业分布饼图、企业数量/占比/注册资本分布表
- 近3年新注册趋势图
- 异常/待人工复核企业清单

**🌟 第二章·主导产业**
- 4维度加权评分模型：
  - 集聚度（企业数量 × 增速）× 25%
  - 科创实力（C022评分 × 专利量）× 30%
  - 企业质量（专精特新/独角兽/上市数）× 25%
  - 资本活力（融资企业数 × 融资总额）× 20%
- 选定综合得分Top 1-2产业，说明选择理由
- 每个主导产业输出：雷达图 + 产业链上/中/下游分布图 + 画像卡

**🔍 第三章·对标分析**
- 选取1-2个全国同类园区（优先选已有公开数据的标杆园区）
- 5维量化对标：规模 / 结构 / 质量 / 生态 / 政策
- 差距识别：逐子环节列出具体差距（含数值）
- 补链机会点：明确薄弱/缺失/高价值环节

**🔗 第四章·强链补链延链**
- 针对每个主导产业分别给出：
  - 强链：巩固现有优势的具体措施
  - 补链：需招引的目标企业类型（列举具体细分赛道）
  - 延链：高附加值下游应用/衍生服务拓展方向
- 每条建议附可执行的政策工具建议

**🎯 第五章·精准招商清单**
- Top50目标企业表格，字段：
  - 企业名称 / 所在城市 / 所属环节 / 科创能力评分 / 匹配度评分（百分制）/ 推荐理由
  - 招商优先级：S级（立即接洽）/ A级（重点跟进）/ B级（长期观察）
- 匹配度评分权重：40%产业链互补性 + 30%选址扩产可能性 + 20%天津产业配套 + 10%政策契合度
- **北京高校科技成果转化专题**（独立子章节）：
  - 重点覆盖：清华大学、北京大学、北航、中国科学院、北京理工大学、北邮等
  - 每所高校卡片包含：核心技术储备方向 / 成果转化机制 / 代表转化案例 / 与园区对接机会点
  - 筛选原则：结合园区主导产业方向，聚焦相关院系

## HTML报告设计规范

### 整体风格
- 深蓝专业色系，适合政务/招商场景
- 紧凑排版，便于领导快速阅读
- 使用 Chart.js CDN 绘制动态图表

### CSS 核心参数（紧凑领导版）
```css
body { font-size: 13px; }
.cover { padding: 40px; }
.section { margin-bottom: 24px; }
.card { padding: 16px; }
.kpi-value { font-size: 24px; }
/* 主色 */
--primary: #1a3a5c;
--accent: #2563eb;
--green: #10b981;
```

### 封面规范
- 标题、副标题、元信息：**居中对齐**
- 封面元信息保留4项（删除"分析方法"）：
  1. 数据截止日期
  2. 报告生成日期
  3. 委托方
  4. 执行框架

### 章节标题规范
- **居中对齐**
- emoji + 章节编号 + 标题在**同一行**横排
- 示例：
```html
<div class="chapter-band" style="text-align:center;">
  🏭 第一章·产业全貌
</div>
```

### 图表规范
- 产业分布：饼图（Chart.js doughnut）
- 综合评分：雷达图（Chart.js radar）
- 对标对比：横向柱状图（Chart.js bar）
- 趋势数据：折线图（Chart.js line）

### 输出路径
```
@session/reports/[park_slug]_report.html
```
`park_slug` 为园区名称拼音缩写，例如：
- 天津滨海中关村 → `tianjin_binhai_report.html`
- 苏州工业园 → `suzhou_sip_report.html`
- 上海张江 → `shanghai_zhangjian_report.html`

## 数据置信度说明

| 数据项 | 置信度 | 说明 |
|-------|-------|------|
| 园区企业总数 | ⭐⭐⭐ 高 | 来自官网/管委会公告 |
| 产业分布占比 | ⭐⭐ 中 | 基于公开信息推算，建议C007批量验证 |
| 科创能力评分 | ⭐⭐ 中 | 智慧芽PatSnap模型推算，建议C022实时验证 |
| 独角兽/上市企业数 | ⭐ 低 | 园区级无公开汇总，标注"无公开数据" |
| Top50招商清单 | ⭐⭐ 中 | 基于PatSnap+公开信息，接洽前建议C022复核 |

---

## 注意事项

1. **无公开数据项** 一律标注"无公开数据"，不得填写推测值
2. **企业清单可验证性**：每家企业须提供企业名称 + 关键判断依据（专利数量/融资事件）
3. **数据截止日期** 在封面、页脚、每个数据卡片角标统一标注
4. **报告版本管理**：如需多次修改，按 `_v2`、`_v3` 等后缀命名新版本
