# 公开数据库覆盖评估参考

## 适用范围
本文件记录各主要公开化学生物数据库的覆盖特征、访问方式与数据量级，
供 Step 2 数据量评估使用。

---

## 数据库概览

### 1. ChEMBL（EMBL-EBI）

- **URL**: https://www.ebi.ac.uk/chembl/
- **API**: https://www.ebi.ac.uk/chembl/api/data/
- **数据量级**: 2,400,000+ 化合物，20,000,000+ 活性记录（ChEMBL 34）
- **靶点覆盖**: 15,000+ 蛋白靶点，含 UniProt 交叉引用
- **活性类型**: IC50、Ki、Kd、EC50、%Inhibition、GI50 等
- **检索方式**: REST API（按靶点 ChEMBL ID 或 UniProt ID）、批量下载（SQLite/CSV）
- **热门靶点数据量参考**:
  - PD-L1（CD274，Q9NZQ7）: ~8,000–12,000 条活性记录
  - 4-1BB（CD137，Q07011）: ~1,500–3,000 条活性记录
  - EGFR: ~50,000+ 条（高热度靶点参考上限）
- **更新频率**: 每年 2–3 次大版本更新

### 2. BindingDB

- **URL**: https://www.bindingdb.org/
- **API**: RESTful API 支持按靶点 UniProt 检索
- **数据量级**: 2,900,000+ 结合亲和力测量值，1,200,000+ 化合物
- **特色**: 大量 Kd 和 ITC 数据，结构解析质量高
- **热门靶点数据量参考**:
  - PD-L1: ~2,000–4,000 条
  - 4-1BB: ~500–1,500 条

### 3. PubChem BioAssay

- **URL**: https://pubchem.ncbi.nlm.nih.gov/
- **API**: PUG REST API
- **数据量级**: 1,500,000+ 生物活性测试记录
- **特色**: 大量 HTS（高通量筛选）数据；NCATS 贡献的 PD-1/PD-L1 专项筛选数据
- **注意**: HTS 数据噪声较大，需严格阈值过滤

### 4. RCSB PDB

- **URL**: https://www.rcsb.org/
- **数据量级**: 220,000+ 蛋白质结构
- **PD-L1 相关结构**: 200+ 条目（含配体复合物）
  - 代表性结构: 5JDR（PD-L1与小分子复合物）, 4ZQK, 5N2F
- **4-1BB/4-1BBL 相关结构**: 30+ 条目
  - 代表性结构: 6MGP（4-1BB/4-1BBL 复合物）
- **用途**: 分子对接生成伪标签；结合口袋分析

### 5. ZINC / ZINC22

- **URL**: https://zinc22.docking.org/
- **数据量级**: 2,000,000,000+ 类药分子（ZINC22）
- **用途**: 负样本采样；类药空间覆盖；虚拟筛选基准库
- **许可**: 访问免费；商业使用条款需查阅最新版本

### 6. DrugBank

- **URL**: https://go.drugbank.com/
- **数据量级**: 14,000+ 药物条目
- **特色**: 药物-靶点相互作用、代谢、副作用综合数据
- **许可**: 学术免费；商业使用需授权

### 7. STRING / BioGRID

- **用途**: 蛋白质相互作用网络，用于靶点通路分析与多靶点协同评估
- **许可**: 开放访问（学术）

## 覆盖热度速查（按靶点类别）

| 靶点类别 | ChEMBL 数据量级 | 建模可行性 |
|---|---|---|
| 免疫检查点（PD-1/PD-L1/CTLA-4） | 5,000–15,000 条 | ⭐⭐⭐⭐ 良好 |
| 共刺激靶点（4-1BB/OX40/CD28） | 500–3,000 条 | ⭐⭐⭐ 一般，需专利补充 |
| 激酶（EGFR/VEGFR/BTK） | 20,000–100,000 条 | ⭐⭐⭐⭐⭐ 丰富 |
| GPCR | 10,000–50,000 条 | ⭐⭐⭐⭐⭐ 丰富 |
| 离子通道 | 3,000–20,000 条 | ⭐⭐⭐⭐ 良好 |
| 核受体 | 5,000–30,000 条 | ⭐⭐⭐⭐ 良好 |
| 新兴靶点（KRAS G12C/PROTAC靶点） | 500–2,000 条 | ⭐⭐ 稀缺，强依赖专利数据 |
