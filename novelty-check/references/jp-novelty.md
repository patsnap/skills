# JP Novelty Quick Guide

Use this guide for Japanese novelty conclusions.

## Core Questions

1. Was the invention publicly known, publicly worked, or described in a publication before filing?
2. Is there one reference or disclosure route that covers every claim limitation?
3. Does any prior-filed later-published application create the main risk?
4. Is a limited grace-period argument being asserted?

## Practical Rules

- Keep novelty in a single-reference frame.
- Check public knowledge, public working, publications, and earlier applications separately.
- Compare claim limitations at the same level of specificity used in the claim.
- Flag technical-effect arguments; they usually belong in inventive-step analysis, not novelty.

## Common Traps

- Mixing novelty reasoning with inventive-step reasoning
- Overlooking public working evidence
- Losing track of numerical or conditional limitations
- Assuming an analogous disclosure is enough

## Output Notes

State:

- strongest JP novelty reference
- disclosure route
- missing elements or ambiguity
- final novelty view and confidence
