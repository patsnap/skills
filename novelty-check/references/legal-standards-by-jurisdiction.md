# Legal Standards by Jurisdiction

Read this file first, then load only the jurisdiction-specific guide you need.

## Selection Order

1. Identify the filing or priority date.
2. Identify the target jurisdiction.
3. Select the jurisdiction guide.
4. Apply that guide before writing the final legal conclusion.

If any routing fact is unresolved and could change the outcome, downgrade to `uncertain`.

## Jurisdiction Routing

- United States: read `references/us-102.md`
- China: read `references/cn-novelty.md`
- Europe: read `references/ep-novelty.md`
- Japan: read `references/jp-novelty.md`

For a PCT or WIPO publication, analyze novelty under the national or regional law that matters for the user decision.

## Cross-Jurisdiction Rules

- Novelty is still a single-reference test.
- Public disclosure date is always critical.
- A readable evidence chain matters more than a large candidate list.
- Unsupported reasoning must be labeled `inference`.
- Definitive conclusions require both a locked jurisdiction and a locked critical date.

## Common Differences

- US: check AIA vs pre-AIA timing and possible grace-period issues.
- CN: remember conflicting applications and local novelty exceptions.
- EP: apply strict absolute novelty; grace-period relief is very limited.
- JP: check public knowledge, public working, publication, and prior-filed disclosure routes.
