# TRIZ Advanced Modules

Use advanced modules only when core TRIZ flow cannot produce a robust recommendation.

## 1. Escalation Triggers
Escalate from core flow when one or more conditions hold:
1. Core options remain high-risk or low-feasibility after first ranking.
2. Contradictions span multiple interacting subsystems.
3. Physical contradiction cannot be resolved with straightforward separation.
4. Stakeholders require deeper novelty exploration.

## 2. ARIZ-lite (Practical Version)
Apply this compact sequence:
1. Rebuild mini-problem model around the key harmful interaction.
2. Define ideal result for that interaction.
3. Identify and sharpen the core contradiction.
4. Explore available resources in and around the conflict zone.
5. Generate transformation ideas (field/object/structure/time changes).
6. Stress-test against constraints and expected side effects.

Output requirement:
- include why ARIZ-lite was triggered
- include at least one option not reachable in core quick mapping

## 3. Separation Principles for Physical Contradictions

### Separation in time
Use when opposite states are needed at different phases.
- Example pattern: high rigidity during operation, low rigidity during assembly.

### Separation in space
Use when opposite states can exist in different regions.
- Example pattern: hot core + cool shell.

### Separation by condition
Use when opposite states depend on operating mode.
- Example pattern: high flow only under peak load.

### Separation between system levels
Use when opposite states can be assigned to subsystem vs supersystem.
- Example pattern: local autonomy + global coordination.

## 4. Su-Field (Substance-Field) Analysis Lite
Use when issue is caused by harmful or missing interaction between objects.

Template:
- S1: acting substance
- S2: target substance
- F: field/interaction (mechanical, thermal, electrical, chemical, informational)
- Interaction status: useful / harmful / missing

Transformation directions:
1. introduce or change field type
2. insert intermediary substance
3. split or merge substances
4. convert harmful interaction into useful controlled interaction

## 5. Risk Control for Advanced Ideas
For each advanced option, explicitly list:
- novelty risk
- integration risk
- validation bottleneck
- kill criteria (when to stop the path)

## 6. Exit Rule
Return to normal 3-option ranking after advanced exploration. Do not stay in open-ended ideation.
